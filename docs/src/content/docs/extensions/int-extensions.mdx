# IntExtensions 功能文档

## 概述

[IntExtensions](../../Chet.Utils/Extensions/IntExtensions.cs) 是一个静态扩展类，为 `int` 类型提供了丰富的扩展方法，涵盖数值判断、转换、运算、格式化等功能，旨在简化整数类型的处理操作，提高代码的可读性和便捷性，特别适用于需要进行数值计算和日常数据展示的场景。

## 主要功能模块

### 2. 范围判断

#### IsBetween

```csharp
public static bool IsBetween(this int value, int min, int max)
```
**示例：**
```csharp

bool result1 = value.IsBetween(0, 100); // true
bool result2 = value.IsBetween(60, 100); // false
```

#### IsBetweenExclusive
判断 int 是否在指定范围内（不包含边界）。
```csharp
public static bool IsBetweenExclusive(this int value, int min, int max)
```

#### Clamp
保证 int 在指定范围内，超出则取边界值。
```csharp
public static int Clamp(this int value, int min, int max)
```
**示例：**
```csharp

int value2 = 50;
int value3 = -10;

int result1 = value1.Clamp(0, 100); // 100
int result2 = value2.Clamp(0, 100); // 50
int result3 = value3.Clamp(0, 100); // 0
```

#### ClampMax

#### ClampMin

### 7. 特殊功能

#### ToRomanString
将 int 转换为罗马数字字符串（1~3999）。
```csharp
public static string ToRomanString(this int value)
```
**示例：**
```csharp
int value1 = 1;
int value2 = 10;
int value3 = 2023;

string result1 = value1.ToRomanString(); // "I"
string result2 = value2.ToRomanString(); // "X"
string result3 = value3.ToRomanString(); // "MMXXIII"
```

#### Repeat
重复执行指定操作。
```csharp
public static void Repeat(this int value, Action action)
public static void Repeat(this int value, Action<int> action)
```
**示例：**
```csharp
int count = 5;

// 无索引版本
count.Repeat(() => Console.WriteLine("Hello"));

// 带索引版本
count.Repeat(i => Console.WriteLine($"第 {i + 1} 次"));
```

#### Times
生成从 0 到 value-1 的整数序列。
```csharp
public static IEnumerable<int> Times(this int value)
```
**示例：**
```csharp
int count = 5;
var sequence = count.Times(); // [0, 1, 2, 3, 4]
```

#### Range
生成从 value 开始的整数序列。
```csharp
public static IEnumerable<int> Range(this int start, int count)
```
**示例：**
```csharp
int start = 10;
var sequence = start.Range(5); // [10, 11, 12, 13, 14]
```

#### ToEnum
将 int 转换为枚举类型。
```csharp
public static TEnum ToEnum<TEnum>(this int value, TEnum defaultValue = default) where TEnum : struct, Enum
```
**示例：**
```csharp
enum Status { Active = 1, Inactive = 2 }

int value = 1;
Status status = value.ToEnum<Status>(); // Status.Active
```

## 使用场景

1. **数值计算** - 进行各种数学运算和比较
2. **金额展示** - 将数值转换为指定格式的字符串显示
3. **数据验证** - 数值范围和状态判断
4. **财务计算** - 货币格式显示、中文大写金额转换
5. **类型转换** - int 与其他数值类型的安全转换
6. **业务逻辑** - 数值状态判断（正负、奇偶、质数等）
7. **循环操作** - 简化循环执行操作
8. **数字显示** - 罗马数字、中文数字转换
9. **枚举处理** - 整数与枚举类型的相互转换

## 注意事项

1. 所有方法都是扩展方法，需要通过 `int` 实例来调用
2. 除法方法提供安全版本 `DivideSafe`，避免除零异常
3. 货币格式方法支持多语言区域设置，默认为中文区域
4. 中文大写金额转换实现了完整的金额格式转换
5. 罗马数字转换仅支持 1~3999 范围内的整数
6. 进制转换方法对超出范围的值返回原值字符串
7. 重复执行方法对无效值（null 或非正数）进行了安全处理
8. 枚举转换方法在枚举值未定义时返回默认值
9. 数学运算方法使用系统内置的转换函数，性能较好
10. 格式化方法支持标准的 .NET 格式字符串
